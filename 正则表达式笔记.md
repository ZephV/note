## 正则表达式格式
 / 规则 / 检索方式

## 检索方式
检索方式,默认是只寻找一个匹配内容的.
i 不区分大小写检索
g 全局检索,找寻所有匹配的内容
可以同时填写两个检索方式即 gi

## js中RegExp对象及其方法
RegExp对象:
```
new RegExp( 规则 , 检索方式 );

```
test( ) 指定字符串是否含有一个指定内容,结果返回boolean值
``` 
var r = new RegExp("e");
result=r.test("i have a pen"); //result等于true.
result = r.test("i am pig"); //result 等于false

```
.exec( ) 指定字符串是否有一个指定的内容,如果有结果返回指定的内容,没有则返回null.

## RegExp的规则
- 方括号
[abc]查找方括号内的任意字符
[^abc]查找除了括号内的字符
[0-9],[a-z],[A-Z],[A-z],[0-9a-zA-Z]第一个是查找任何0到9的字符,其他同理
- 元字符
. 代表某个字符,除了换行符和行结束符
\w 查找所有[0-9A-z]的字符 \W 查找除[0-9A-z]以外的字符
(根据规律,\字母大写就是查找\字母小写规则相反的内容)
\d 找数字 \D
\s 找空格 \S
\b 查找字符边界 ( 检索"Apple"规则/le\b/g 返回le,规则/\ble/返回null ) \B
\n 查找换行字符的位置 结果显示换行字符的位置
- 量词
n+ 查找至少一个n的字符串
n* 查找0个或多个n的字符串
n? 查找0或1个n的字符串
n{x} 查找有x个n的字符串
n{x,y} 查找含有x~y个n的字符串
n{x,} 查找至少x个n的字符串
n$ 以n结尾的字符串
^n 任何以n开头的字符串
?=n 任何字符串后紧接着n的(n不在此字符串内)
?!n 后面字符串后不紧接着n的

## 支持正则表达式的String方法
search 检索与正则表达式匹配的值
match 查找正则表达式的匹配
replace 替换正则表达式匹配的子串
split 把字符串分割为字符数组
